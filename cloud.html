Security in the Cloud:

Security Overview, Cloud Security Challenges and Risks, Softwareas-a Service Security, Security Governance, Risk Management, Security Monitoring, Security Architecture Design, Data Security, Application Security, Virtual Machine Security, Identity Management and Access Control, Autonomic Security
